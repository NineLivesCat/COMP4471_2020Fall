# COMP4471_2020Fall

## Description
This course project for HKUST COMP4471/ELEC4240 2020 fall semester.  
Project members:  
CHEN, Sida (schenci@connect.ust.hk)  
HUANG, Qiucan (qhuangag@connect.ust.hk)  
LIN, Yifei (ylinbz@connnect.ust.hk)
